# My 5 Shortcuts — PopClip Extension

This extension adds **5 buttons** (with configurable icons) to the PopClip bar. Each button sends a **keyboard shortcut** that you define in the extension settings. For each button you can also pick an **icon** from a long list (copy, scissors, screenshot, OCR, screen recording, speaking bubble, etc.).

## Installation

1. In Finder, **double‑click** the `CustomShortcuts.popclipext` folder.
2. PopClip will install the extension (you may see a warning about an unsigned extension — you can approve it).
3. In PopClip go to **Actions** and enable the **“My 5 Shortcuts”** extension.

## Configuring shortcuts and icons

1. In PopClip settings: **Actions** → find **“My 5 Shortcuts”** → click the **gear icon** (options).
2. For each button (1–5) set:
   - **Shortcut** — key combination (e.g. `command b`, `option shift s`).
   - **Icon** — choose from the list (numbers, copy, scissors, screenshot, OCR, recording, speech bubble, etc.).
3. Shortcut format (type as text, don’t press the keys):
   - `command b` — ⌘B  
   - `command shift i` — ⌘⇧I  
   - `option space` — ⌥Space  
   - `control option command x` — ⌃⌥⌘X  

**Modifiers:** `command`, `option`, `control`, `shift` (can be combined).  
**Keys:** letter (e.g. `b`), `return`, `space`, `escape`, `delete`, `tab`, arrow keys, F1–F20, etc.

## Usage

Select any text → the PopClip bar appears with buttons 1–5 → tap a button to “press” its shortcut in the active app.
